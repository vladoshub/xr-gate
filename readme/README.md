# XR Tracking documentation

This folder contains practical notes for optional subsystems and device-specific workflows.

## Files

- `xreal_ultra/README.md` — XREAL Ultra build/run guide for Linux and Windows package flows.
- `xrizer.md` — optional xrizer build/run notes. xrizer is GPL-3.0-or-later upstream and is not built by default.
- `override_controller.md` — quick notes for controller/keyboard input override.
