# Override controller

Use Bluetooth controllers, USB keyboards, or joysticks as runtime controller input.

## Run

```bash
cd ~/src/xr_tracking
out/xreal_ultra/bin/scripts/override_controller/start_override_controller.sh
```

Finish the training flow in the terminal.

For controller-only input, use `controller_buttons_only` or the relevant controller input mode in the runtime adapter configuration.
